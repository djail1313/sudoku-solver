# sudoku-solver
Solving Sudoku problem. Using Assembly x86 Linux

Requirement:
- GCC
- Fedora 23, 64bit need to install
  - glibc-devel.i686 libgcc.i686
- Ubuntu 15, 64bit need to install
  - gcc-multilib

How to Build & Run:
- #make
- ./main
